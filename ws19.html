    • Windows Server 2019
    Introduction

    In questo spazio passiamo in rassegna Windows Server 2019. In dettaglio nelle immagini sottostanti parliamo di come installare nuove funzioni e monitorare il server ed infine come schedulare un task indicando quando eseguirlo.

    Per installare nuovi componenti aprire il Server Manager e selezionare Add Roles and Features poi selezionare i componenti che si desidera rendere disponibili. In questo esempio vengono selezionati i servizi DHCP ed SMTP. Seguire il menu ed attendere che l' installazione sia completa. Dalla Dashboard è sempre possibile verificare i servizi installati ed eventualmente installarne altri oppure modificarne i parametri. Inoltre è possibile controllare gli eventi e lo stato del server.
    A questo punto l'installazione è terminata ed è possibile verificare nella Dashboard lo stato del sistema e l'attivazione dei nuovi componenti.
